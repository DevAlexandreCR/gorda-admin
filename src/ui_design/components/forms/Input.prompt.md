One-liner: Standard Gorda form field — bordered input, 700-weight label, pink focus ring matching the brand.

```jsx
<Input label="Email Address" type="email" placeholder="Enter email" icon="fas fa-envelope" />
<Input label="Phone" placeholder="Enter phone" error="Phone is required" />
```

Pass `icon` for a leading Font Awesome glyph, `error` to enter the invalid state. Disabled inputs use the gray-200 fill.
